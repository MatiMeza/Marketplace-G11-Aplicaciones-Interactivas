# Marketplace-G11-Aplicaciones-Interactivas
## Documentación de endpoints definidos

### Recurso: Joyas

- **GET /joyas**  
  Obtiene el listado completo de joyas.

- **GET /joyas/{id}**  
  Obtiene una joya específica a partir de su ID.

- **POST /joyas**  
  Permite crear una nueva joya.

---

### Recurso: Materiales

- **GET /materiales**  
  Obtiene el listado completo de materiales.

- **GET /materiales/{id}**  
  Obtiene un material específico a partir de su ID.

- **POST /materiales**  
  Permite crear un nuevo material.
